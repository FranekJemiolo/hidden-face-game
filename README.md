# Hidden Face - 3D Cube Puzzle

A deterministic 3D cube puzzle game with solvable scrambles and shareable puzzles.

## Features

- **Deterministic Seeding**: Same seed always generates the same puzzle
- **Solvable Puzzles**: All puzzles are guaranteed solvable by construction (scrambled from solved state)
- **URL Sharing**: Share any puzzle via URL with seed and scramble encoding
- **3D Slice Animations**: Smooth CSS-based 3D rotation animations
- **Camera Control**: Rotate around the cube using arrow keys
- **Move Tracking**: Track your moves to solve the puzzle

## How to Play

1. **Rotate Slices**: Click and drag on any tile to rotate the corresponding slice
   - Horizontal drag: rotates Y-axis layer
   - Vertical drag: rotates X-axis layer
2. **Camera Control**: Use left/right arrow keys to rotate the camera around the cube
3. **Share Puzzle**: Click "Share Puzzle" to copy the puzzle URL to clipboard
4. **Goal**: Return the cube to its solved state

## URL Format

```
https://franekjemiolo.github.io/hidden-face-game/#seed=12345&scramble=X0+1,Y2-1,X1+1
```

- `seed`: Random seed for deterministic generation
- `scramble`: Comma-separated list of moves (Type, Layer, Direction)

## Technical Details

### Core Systems

- **Seeded RNG**: LCG (Linear Congruential Generator) for deterministic random numbers
- **Cube State**: 3×3×3 array with color-coded cells
- **Rotations**: Pure mathematical transformations (X and Y axis)
- **Camera**: 4-position rotation around the cube
- **Face Mapping**: Camera-aware projection of visible face

### Solvability Guarantee

All puzzles are generated by:
1. Starting from solved state
2. Applying valid moves only (X/Y axis rotations)
3. Recording the move sequence

This ensures every puzzle is in the valid permutation group and solvable.

### Animation System

- DOM-based 3D transforms using CSS
- Slice-level animation with preserve-3d
- 300ms cubic-bezier easing for natural feel
- No WebGL required
